Entendendo Comunicação Client x Server

(Voltar)

A comunicação Cliente-Servidor é um modelo onde um dispositivo chamado "cliente"solicita recursos ou serviços a outro chamado "servidor". O cliente envia requisições, e o servidor responde fornecendo os dados ou executando a ação solicitada.

Os Primeiros Computadores

(Voltar)

Os primeiros computadores surgiram no século XX. Eram máquinas enormes, usadas principalmente para cálculos científicos e militares. O ENIAC, construído em 1946, é considerado um dos primeiros computadores eletrônicos.

Como Surgiu a internet?

(Voltar)

A internet nasceu nos anos 1960, como um projeto militar nos EUA chamado ARPANET. Evoluiu para conectar universidades e organizações, crescendo globalmente. O World Wide Web (WWW) revolucionou a internet nos anos 1990.

Cliente no Linguajar Web

(Voltar)

No contexto web, o "cliente" refere-se ao navegador ou aplicação que acessa e exibe informações. Ele envia solicitações a servidores para obter dados ou executar ações.

Navegadores

(Voltar)

Navegadores são aplicativos que permitem acessar e interagir com conteúdo na web. Exemplos incluem Chrome, Firefox e Safari.

Aplicações Web

(Voltar)

São programas acessados através de navegadores. Exemplos incluem redes sociais, e-mail e ferramentas de produtividade.

Dispositivos Móveis

(Voltar)

Refere-se a smartphones e tablets. Têm capacidade de navegar na internet e executar aplicativos.

Sobre Servidores

(Voltar)

Servidores são computadores especializados que respondem a solicitações de clientes. Armazenam dados, executam aplicativos e gerenciam a comunicação.

Hardwares e Softwares em Servidores

(Voltar)

Hardware são os componentes físicos dos servidores. Software são os programas que os controlam.

Tipos de Servidores

(Voltar)

Existem servidores de arquivos, de banco de dados, de aplicativos, etc., cada um com uma função específica.

Hospedagem de Sites

(Voltar)

Hospedagem de sites é o serviço que possibilita armazenar e tornar acessível um site na internet.

Server-Side

(Voltar)

Refere-se a processamento que ocorre no servidor. Dinâmico, gera conteúdo antes de enviar para o cliente.

Client-Side

(Voltar)

Refere-se a processamento no navegador do cliente. Dinâmico, manipula conteúdo após recebê-lo.

Introdução ao HTML na Prática

(Voltar)

HTML é a base para criar páginas web. Utiliza tags para estruturar o conteúdo.

HTML não é Linguagem de Programação

(Voltar)

HTML (Hypertext Markup Language) é uma linguagem de marcação usada para estruturar conteúdo na web, mas não executa lógica ou tarefas complexas como uma linguagem de programação.

Estrutura Básica do HTML

(Voltar)

Todo documento HTML tem uma estrutura básica com elementos como "html", "head", "title" e "body".

Falando sobre Tags

(Voltar)

Tags são elementos HTML como "p" para parágrafos ou "img" para imagens.

Atributos das Tags

(Voltar)

Atributos fornecem informações extras para as tags, como o atributo src em uma tag "img".

Textos

(Voltar)

Tags como "h1" a "h6" são usadas para cabeçalhos. "p" para parágrafos.

Listas Ordenadas e Não Ordenadas

(Voltar)

"ul" para listas não ordenadas, "ol" para ordenadas. "li" define cada item da lista.
